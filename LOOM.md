# Loom Operating Context

This file is a freeform scratchpad included in agent context.

Human + AI edits are expected.

## Compound baseline

- Core loop: Plan -> Work -> Review -> Compound -> Repeat.
- Automatic compounding path is observations -> instincts.
- Skill authoring is explicit and harness-native.

## Compound files

- Observations log: `.loom/compound/runtime/observations.jsonl`
- Instinct store: `.loom/compound/instincts.json`
- Instinct index: `.loom/compound/INSTINCTS.md`
- Roadmap scratchpad: `.loom/compound/ROADMAP.md`

## Command shortcuts

- `/loom-plan` -> create/refine tickets and plan
- `/loom-work` -> implement in worktree
- `/loom-review` -> review before merge
- `/loom-compound` -> capture durable learnings (instincts + memory)
