# Constitutional Writing Standard

Good constitutional writing should answer:

- what future work is allowed
- what future work is encouraged
- what future work is constrained
- what future work is ruled out

## Anti-patterns

Avoid:

- vague philosophy with no downstream consequence
- ticket-like progress notes
- speculative behavior detail that belongs in a spec
- sequencing detail that belongs in a plan

## Review Questions

Before you accept a constitutional update, ask:

1. would a future agent know how this changes judgment
2. is the record durable rather than reactive
3. is the consequence of the decision explicit
4. would the wrong downstream layer have been tempted to hold this truth if constitution did not
